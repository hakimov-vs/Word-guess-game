# Word Guess Game

## Overview
The Word Guess Game is a console-based application that allows users to guess a word based on asterisks representing the letters of the word. Players will be prompted to enter their guesses and will receive feedback on their attempts, making the game both fun and educational.

## How to Play
1. The game will display a series of asterisks, each representing a letter in the secret word.
2. Players will guess letters one at a time.
3. If a guessed letter is in the word, it will replace the corresponding asterisk(s).
4. If the guessed letter is not in the word, the player will be informed and prompted to guess again.
5. The game continues until the player correctly guesses the word or exhausts their chances.

## Features
- Console-based gameplay
- User-friendly interface
- Feedback on guesses
